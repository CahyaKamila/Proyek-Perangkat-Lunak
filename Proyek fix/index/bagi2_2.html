Metode Bagi Dua

Contoh:

Tentukan akar positif terbesar dari persamaan

\(f(x) = 3\) \(sin(x) — x + 1 = 0\)

Hitung dengan toleransi \(0.001\).


Penyelesaian:

Berdasarkan contoh di atas, dapat diketahui bahwa akar terbesar dari persamaan tersebut berada pada selang \((2\) , \(3)\).


Iterasi 1: Selang Akar \((2\) , \(3)\)

\(a = 2\)

\(b = 3\)

\(f(a)\) -> \(f(2) = 3\) \(sin(2) — 2 + 1 = 1.72789\)

\(f(b)\) -> \(f(3) = 3\) \(sin(3) — 3 + 1 = -1.57664\)

\(T=\frac{2+3}{2} = 2.5\)

\(f(T)\) -> \(f(2.5) = 3\) \(sin(2.5) — 2.5 + 1 = 0.29542\)

\(f (a).f (T)\) -> \((1.72789\) . \(0.29542) > 0,\)
maka \(a = 2.5\) , selang baru \((2.5\) , \(3)\)

\(|2.5 — 3| = 0.5 > Tolerasi\), lanjut iterasi \(2\)


Iterasi 2: Selang Akar \((2.5\) , \(3)\)

\(a = 2.5\)

\(b = 3\)

\(f(a)\) -> \(f(2.5) = 3\) \(sin(2.5) — 2.5 + 1 = 0.2954164\)

\(f(b)\) -> \(f(3) = 3\) \(sin(3) — 3 + 1 = -1.57664\)

\(T=\frac{2.5+3}{2} = 2.75\)

\(f(T) = 3\) \(sin(2.75)- 2.75 + 1 = -0.605017\)

\(f (a).f (T)\) -> \((0.2954164\) . \(-0.605017) < 0,\) maka
\(b = 2.75\), selang baru \((2.5, 2.75)\)

\(|2.5 — 2.75| = 0.25 > Tolerasi\), lanjut iterasi \(3\)


Iterasi 3: Selang Akar \((2.5\) , \(2.75)\)

\(a = 2.5\)

\(b = 2.75\)

\(f(a)\) -> \(f(2.5) = 3\) \(sin(2.5) — 2.5 + 1 = 0.2954164\)

\(f(b)\) -> \(f(2.75) = 3\) \(sin(2.75) — 2.75 + 1 = -0.605017\)

\(T=\frac{2.5+2.75}{2} = 2.625\)

\(f(T) = 3\) \(sin(2.625)- 2.625 + 1 = -0.1432491\)

\(f (a).f (T)\) -> \((0.2954164\) . \(-0.1432491) < 0,\) maka
\(b = 2.625\), selang baru \((2.5, 2.625)\)

\(|2.5 — 2.625| = 0.125 > Tolerasi\), lanjut iterasi \(4\)
